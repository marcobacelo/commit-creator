# 🧰 Pré-requisitos

---

## 📌 O que você precisa ter instalado

Para compilar, instalar e usar o **Commit Pattern CLI**, você precisa ter:

* 🦀 Rust
* 📦 Cargo
* 🌳 Git
* 🖥️ Terminal compatível com `bash`, `zsh` ou `fish`

---

## 🦀 Verificar Rust

```bash
rustc --version
```

Exemplo de saída:

```txt
rustc 1.XX.X
```

---

## 📦 Verificar Cargo

```bash
cargo --version
```

Exemplo de saída:

```txt
cargo 1.XX.X
```

---

## 🌳 Verificar Git

```bash
git --version
```

Exemplo de saída:

```txt
git version 2.XX.X
```

---

## 🖥️ Verificar shell em uso

```bash
echo $SHELL
```

Exemplos comuns:

```txt
/bin/bash
/bin/zsh
/usr/bin/fish
```

---

## ✅ Checklist

Antes de continuar, confirme:

* [ ] Rust instalado
* [ ] Cargo instalado
* [ ] Git instalado
* [ ] Projeto clonado localmente
* [ ] Terminal aberto no diretório do projeto
